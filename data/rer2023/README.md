# RER2023-derived data

This folder contains derivative data and experimental outputs generated from the RER2023 landslide polygon inventory for the single-layer landslide-polygon reconstruction and downstream feature-table experiments.

The original RER2023 shapefile is not redistributed in this repository because its redistribution terms should be checked from the original data provider. Users who wish to reproduce the raw-data preprocessing step should obtain the original RER2023 landslide inventory from its official source and cite it according to the corresponding data-use terms.

The files provided here include processed single-layer SDF-corner-mask feature databases, latent reconstruction results, reconstruction evaluation outputs, and latent feature tables used in the manuscript.
